## 中英爱永恒A款分红边界值汇总

- 从 `费率表.xlsx`（Sheet: 基本保险金额对应费率表）提取边界值费率
- 费率表结构：行1=产品名称, 行5=标准体, 行6=保障方案(方案一/方案二), 行7=保险期间(终身), 行8=交费期间, 行9=性别, 行10+ =年龄数据
- 生成 `中英爱永恒A款分红边界值汇总.xlsx`:
  - 57条边界值记录
  - 2种保障方案（方案一、方案二）× 7种交费期间（趸交/3/5/10/15/20/30年）× 2种性别（男、女）
  - 交费期间对应最大年龄：趸交=69, 3年=65, 5年=62, 10年=62, 15年=60, 20年=59, 30年=50
  - ensurePlan 固定为 "1"（仅标准体，无优选体）
  - 只有1个Sheet（不同于瑞泰鸿利的2个Sheet）
